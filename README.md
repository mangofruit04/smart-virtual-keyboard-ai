# ✋ Smart Virtual Keyboard using Hand Gestures

A Computer Vision-based Smart Virtual Keyboard developed in **Python** that enables users to type without a physical keyboard using real-time hand gestures. The application uses **OpenCV**, **CVZone**, and **MediaPipe** to track hand movements and detect pinch gestures for typing. It also features intelligent word suggestions, making text input faster and more intuitive.

---

## 🚀 Features

- ✋ Real-time hand tracking
- 🤏 Pinch gesture detection for typing
- ⌨️ Fully functional virtual keyboard
- 💡 Intelligent word suggestion system
- 🗑️ Delete, Clear, and Space keys
- 🎨 Interactive UI with hover and click animations
- 📊 Live FPS monitoring
- 📷 Webcam-based interaction (No additional hardware required)

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/mangofruit04/Smart-Virtual-Keyboard.git
```

### Navigate to the project folder

```bash
cd Smart-Virtual-Keyboard
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python main.py
```

## 🎮 How It Works

1. Launch the application.
2. Allow access to your webcam.
3. Move your index finger over a key.
4. Pinch your thumb and index finger together to press the selected key.
5. Use the built-in word suggestions for faster typing.
6. Press **Space**, **Delete**, or **Clear** using gestures when needed.
